# METRONIC DIRECTORY BUILDER - BLUEPRINT COMPLETO ## Fase 2: Implementação da Página de Construção de Directories ### TechSites.ai | Directory Factory | Maio 2026 --- ## 1. VISÃO GERAL Este documento contém o blueprint completo para implementar a página **Directory Builder** no painel Metronic. A página permite construir um directory completo (de 0 a LIVE) em menos de 30 minutos, totalmente automatizado via workflows N8N. **URL da página:** `/admin/directory-builder` **Funcionalidades:** - Formulário estruturado em 6 blocos temáticos - Validação client-side completa (JavaScript) - Trigger de webhook N8N com payload JSON estruturado - Polling em tempo real do status de construção - Barra de progresso por micro tarefa (MT-01 a MT-11) - Log de execução em tempo real - Notificação de sucesso com URL do directory LIVE --- ## 2. LÓGICA JAVASCRIPT ### 5.1. Validação do Formulário ```javascript function validateForm() { const errors = []; // Validação Bloco 1 if (!$('#project_name').val() || $('#project_name').val().length < 3) { errors.push('Nome do projeto é obrigatório (mín 3 caracteres)'); } if (!$('#domain').val()) { errors.push('Domínio raiz é obrigatório'); } const subdomain = $('#subdomain').val(); if (!subdomain || !/^[a-z0-9-]{2,63}$/.test(subdomain)) { errors.push('Subdomínio inválido (apenas a-z, 0-9 e hífen, 2-63 chars)'); } // Validação Bloco 2 const listingsCount = parseInt($('#listings_count').val()); if (isNaN(listingsCount) || listingsCount < 10 || listingsCount > 10000) { errors.push('Número de listings deve estar entre 10 e 10.000'); } // Validação Bloco 5 if (!$('#github_repo').val() || !/^[a-zA-Z0-9-_]+$/.test($('#github_repo').val())) { errors.push('Nome do repositório GitHub inválido'); } const zoneId = $('#cloudflare_zone_id').val(); if (!zoneId || !/^[a-f0-9]{32}$/.test(zoneId)) { errors.push('Cloudflare Zone ID inválido (32 caracteres hexadecimais)'); } if (errors.length > 0) { showErrors(errors); return false; } return true; } ``` ### 5.2. Construção do Payload ```javascript function buildPayload() { return { project_name: $('#project_name').val(), domain: $('#domain').val(), subdomain: $('#subdomain').val(), full_domain: $('#full_domain').val(), niche: $('#niche').val(), language: $('#language').val(), listings_count: parseInt($('#listings_count').val()), listings_source: { type: $('#listings_source_type').val(), gdrive_folder_id: $('#gdrive_folder_id').val() || null, csv_url: $('#csv_url').val() || null }, categories: $('#categories').val().split(',').map(c => c.trim()), blog: { posts_informational: parseInt($('#posts_informational').val()) || 0, posts_amazon_reviews: parseInt($('#posts_amazon_reviews').val()) || 0, posts_local_reviews: parseInt($('#posts_local_reviews').val()) || 0, keywords: $('#keywords_seo').val().split(',').map(k => k.trim()) }, monetization: { amazon_affiliate_id: $('#amazon_affiliate_id').val() || null, google_adsense_id: $('#google_adsense_id').val() || null, google_analytics_id: $('#google_analytics_id').val() || null, listings_premium_price: parseFloat($('#listings_premium_price').val()) || 0, restro_saas_enabled: $('#restro_saas_enabled').is(':checked') }, infrastructure: { github_repo: $('#github_repo').val(), github_owner: 'reynaldodallin', cloudflare_zone_id: $('#cloudflare_zone_id').val(), gdrive_folder_id: $('#gdrive_backup_folder_id').val() || null, n8n_webhook_url: 'https://n8n.waas.host/webhook/directory-factory' }, timestamp: new Date().toISOString(), triggered_by: 'admin@techsites.ai' // Substituir por usuário logado }; } ``` ### 5.3. Trigger do Webhook N8N ```javascript function triggerBuild() { if (!validateForm()) return; const payload = buildPayload(); const webhookUrl = 'https://n8n.waas.host/webhook/directory-factory'; // Mostrar modal de progresso $('#progressModal').modal('show'); $('#buildButton').prop('disabled', true).text('Construindo...'); // Enviar para N8N fetch(webhookUrl, { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify(payload) }) .then(response => response.json()) .then(data => { // Iniciar polling de status const buildId = data.build_id; startPolling(buildId); }) .catch(error => { console.error('Erro ao disparar build:', error); showError('Falha ao iniciar construção do directory. Tente novamente.'); $('#buildButton').prop('disabled', false).text('Construir Directory'); }); } ``` ### 5.4. Polling de Status ```javascript let pollingInterval; let startTime = Date.now(); function startPolling(buildId) { updateTimer(); pollingInterval = setInterval(() => { fetch(`https://n8n.waas.host/webhook/directory-factory/status/${buildId}`) .then(response => response.json()) .then(status => { updateProgress(status); if (status.build_status === 'completed') { clearInterval(pollingInterval); showSuccess(status); } else if (status.build_status === 'failed') { clearInterval(pollingInterval); showError(status.error_message); } }) .catch(error => { console.error('Erro no polling:', error); }); }, 3000); // Poll a cada 3 segundos } function updateTimer() { setInterval(() => { const elapsed = Math.floor((Date.now() - startTime) / 1000); const minutes = Math.floor(elapsed / 60); const seconds = elapsed % 60; $('#timerDisplay').text(`${minutes}m ${seconds}s`); }, 1000); } function updateProgress(status) { // Atualizar barra de progresso geral const progress = (status.completed_microtasks.length / 11) * 100; $('#progressBar').css('width', progress + '%').text(Math.floor(progress) + '%'); // Atualizar status atual $('#currentStatus').text(status.current_microtask_name || 'Preparando...'); // Atualizar checkmarks das micro tarefas status.completed_microtasks.forEach(mt => { $(`#${mt}-check`).removeClass('text-muted').addClass('text-success'); $(`#${mt}-check i`).removeClass('fa-circle').addClass('fa-check-circle'); }); // Adicionar ao log if (status.last_log_entry) { const timestamp = new Date().toLocaleTimeString(); $('#executionLog').append(`[${timestamp}] ${status.last_log_entry}\n`); $('#executionLog').scrollTop($('#executionLog')[0].scrollHeight); } } function showSuccess(status) { $('#progressModal').modal('hide'); Swal.fire({ icon: 'success', title: 'Directory Construído com Sucesso!', html: `

Seu directory está LIVE em:

${status.live_url}

Tempo total: ${$('#timerDisplay').text()}

`, confirmButtonText: 'Ver Directory' }).then(() => { window.open(status.live_url, '_blank'); }); $('#buildButton').prop('disabled', false).text('Construir Novo Directory'); } function showError(message) { Swal.fire({ icon: 'error', title: 'Erro na Construção', text: message, confirmButtonText: 'Tentar Novamente' }); $('#buildButton').prop('disabled', false).text('Construir Directory'); } ``` --- ## 6.
